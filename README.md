# Tu salario en pesos (ARG) convertido a varios dólares
<a name="up"></a>
El objetivo de esta app es permitir ver tu salario en diferentes valores de dólares. A saber:  
-Dólar Oficial  
-Dólar Blue  
-Dólar Turista  
-Dólar Bolsa  
-Dólar Mayorista  
-Dólar Contado con Liquidación
